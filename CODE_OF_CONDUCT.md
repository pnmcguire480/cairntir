# Code of Conduct

Cairntir adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Assume good faith.
- Disagree with ideas, not people.
- No harassment, no discrimination, no personal attacks.
- Maintainers will enforce this consistently.

## Scope

This Code of Conduct applies in all project spaces (GitHub issues, PRs, discussions, Discord if/when one exists) and when an individual is representing the project in public spaces.

## Reporting

Report violations to the project owner: open a private issue draft and tag `@pnmcguire480`, or email via the address listed on the GitHub profile.

All reports are handled confidentially. Retaliation against reporters is itself a violation.

## Enforcement

Maintainers will follow the [Contributor Covenant Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines): correction → warning → temporary ban → permanent ban, escalating with severity.
